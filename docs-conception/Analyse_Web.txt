CONNEXION API
- - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - -

SESSIONS
Lister les sessions => get api/v1/sessions/list
Ajouter une session => post api/v1/sessions/add
Récupérer le montant collecté pour une session => get api/v1/sessions/get-actual-amount/:id
Calculer et mettre à jour le montant récolté => => put api/v1/sessions/update-amount/:id

PLAYERS
Ajouter un participant dans une session
Mettre à jour le score d'un participant
Récupérer les participants d'une session avec leur score
Récupérer le score d'un participant à une session

QUIZ
Lister les quiz
Récupérer les questions d'un quiz
Récupérer les réponses d'une question
Enregistrer la réponse donnée par un participant à une question lors d'une session
